### Flutterのinstall

- [このリンク](https://flutter.dev/docs/get-started/install/windows)からダウンロード
- 任意のフォルダに解凍
  - 例えば、`c:\env`にフォルダを作成しその中で解凍する
  - `c:\env\flutter\...`となる
- 環境変数`path`に追加する
  - 例の場合なら`c:\env\flutter\bin`
- コマンド`flutter doctor`が実行できればOK
